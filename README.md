RR Castings — Gate Pass Management System

A web-based gate pass management system for tracking inward and outward material movement.
---

Access

Live App: https://aashishsurani44.github.io/rr-castings/
Contact the admin for login credentials.
---

User Roles

| Role | What they can do |

| Admin | Full access — create, edit, delete, settings |
| User | Create and edit gate passes, manage customers and products |
| Viewer | View gate passes and dashboard only |
---

Features

- Create inward and outward gate passes
- Auto-generated gate pass numbers
- Multi-item support per gate pass
- View, edit and delete gate passes
- Filter gate passes by type, month, year, date range, customer and gate pass number
- Download filtered gate passes as Excel with color coding
- Dashboard with monthly stats and recent gate passes
- Customer and product master management
- Role-based access control
- Real-time data updates across all users
- Automatic monthly Excel report sent to email on the 1st of every month
---

How to Use:

**Create a Gate Pass**
1. Click **Create Gate Pass** in the sidebar
2. Fill in customer details, add items, vehicle and weighbridge info
3. Click **Save Gate Pass**

**View Gate Passes**
1. Click **View Gate Passes** in the sidebar
2. Click any row to view full details
3. Use the **Filters** button to search and filter
4. Click **Download Excel** to export

**Edit a Gate Pass**
1. Open the gate pass from View Gate Passes
2. Click **Edit** in the view modal
---

Monthly Report

An Excel report is automatically generated on the 1st of every month at 9:00 AM and sent to the registered email addresses. The report covers all gate passes from the previous month and is also saved to Google Drive.
